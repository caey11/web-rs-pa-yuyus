Healthy Love - Klinik sederhana (PHP + MySQL)
--------------------------------------------
Cara pakai:
1. Copy folder ke htdocs (XAMPP) atau gunakan built-in PHP server.
2. Buat database MySQL bernama `healthy_love`.
3. Import tabel yang ada di file `database.sql` menggunakan phpMyAdmin atau CLI.
4. Sesuaikan koneksi di `koneksi.php` jika perlu.
5. Buka browser: http://localhost/healthy_love/index.php
